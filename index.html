    How to install showMe :
  • Insert link to jquery and ShowMe librairies in the head of your web page
  • Assign the class "showMe" at element you want it be open as slideshow
  • showMe grabs the "href" of these elements, analyse them and opens its in slide show
  • Title of links is shown at the bottom of the screen when slideshow is on
  • if attribute "data-link" is filled, a link is put on picture and a click on it send user towards this link
  • For html content, "href"" must end with "showMe_html" and you have to add attribut called "data-showMeRel" which contains id of element which will be displayedn in the slideshow
  • Javascript parameters you can add in your web page :
    • var showMe_backgroundColor='#1e1e1e';
      color of : gradient bottom of ambilight, background of picture with no ambilight,html content and movie background
    • var showMe_ambilight=1;
      0 : No ambilight, background color is equal to showMe_backgroundColor
      1 : background color is tuned with main color of picture
      2 : A nebula atmosphere based on picture is used to fill the background (beautifull but transition will probably be slower)
    • var showMe_backgroundColorGradient=true;
      Background color is graded from 'showMe_backgroundColor' to main color of picture if showMe_ambilight is true
    • var showMe_effect='slide';
      Effect shown between 2 pictures change : 'slide', 'fade' or 'nothing'
    • var showMe_speedEffect='200';
      Speed of effect in milliseconds
    • var showMe_Opacity='1';
      Opacity of background. Let see or not the web page under
    Others informations :
  • Pre-loading elements for faster loading
  • You can make a link directly to a picture if you add "#idPicture" at the end of url : Try it
  • "Right", "left" and "escape" on your keyboard have actions on slideshow
