ShowME

Keyboard, mouse and touch events

Very light and simple to use.

Show source to see how it simply works

    ShowMe open :
  • jpg, jpeg, gif, png and webp (if browser accepts it)
  • mp4 movies
  • html contents
  • pdf
    ShowMe use :
  • Color Thief v2.0 by Lokesh Dhakar (embeded directly in showMe script, you don't need to embed yourself)
  • jquery v2.1.4 (because of version of jqueryMobile) If you don't want to use jquery mobile you can use any jquery version you want. jqueryMobile is only use to trap swipe envents but showMe works on mobile without it.
  • jqueryMobile v1.4.5 (not necessary)
    How to install showMe :
  • Insert link to jquery, jqueryMobile and ShowMe librairies in the head of your web page
  • Assign the class "showMe" at element you want it be open as slideshow
  • showMe grab the "href" of these elements, analyse them and open its in slide show
  • Title of links is shown at the bottom of the screen when slideshow is on
  • For html content, "href"" must end with "showMe_html" and you have to add attribut called "data-showMeRel" which contains id of element which will be displayedn in the slideshow
  • Javascript parameters you can add in your web page :
    • var showMe_backgroundColor='#222222';
      color of : gradient bottom of ambilight, background of picture with no ambilight,html content and movie background
    • var showMe_ambilight=true;
      background color is tuned with main color of picture
    Others informations :
  • Pre-loading elements for faster loading
  • You can make a link directly to a picture if you add "#idPicture" at the end of url : Try it
  • "Right", "left" and "escape" on your keyboard have actions on slideshow
